# Frontend

Interfaz simple para validar el asistente de redacción.

- `index.html`: página web principal.
- `styles.css`: estilos básicos.
- `app.js`: lógica de frontend y llamada a la API.
- `server.js`: servidor estático Node.

Uso:
1. Instala dependencias: `npm install`
2. Inicia el servidor: `npm start`
3. Abre `http://127.0.0.1:5500` en el navegador.
