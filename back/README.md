# Backend

Mini API para cargar el modelo y corregir texto usando Node.js.

- `server.js`: servidor Express.
- `package.json`: dependencias y script de inicio.

Uso:
1. Activa el entorno virtual: `source ../model/.venv/bin/activate`
2. Instala dependencias: `python3 -m pip install -r requirements.txt`
3. Inicia el servidor: `./run_back.sh`
4. Envía texto a `http://127.0.0.1:8000/correct`.
